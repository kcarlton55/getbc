Virtual environments

If you intend to use only getbc.py to install and/or upgrade bomcheck (and bomcheckgui), you do not need to know anything about what this article has to say.  But if you wish to have a bit more control of Python's handling of bomcheck, read on.

What is a virtual environment?

The computer language Python often uses what's called virtual envirnments to isolate code packages in order to keep the code organized and managed.  If you used getbc.py to install bomcheck, a virtual environment (VE) was automatically created for you.  Bomcheck, bomcheckgui, and their dependent code were put into that virtual enviornment.  Only the VE that getbc produced will be discussed in this article. Other means of creating VEs are outside the scope of this article.

Getbc.py produces a virtual environment called bc-venv.  It is essentially just a directory named bc-venv. It contains the bomcheck code plus it contains some additional code that can activate the virtual environment.  When activated, only the code from within that environment is available to execute.  Bc-venv could have been placed anywhere on the drive of your computer, but getbc.py was designed to put bc-venv in your user folder.

Activating the bc-venv virtual environment

Bc-venv contains a subdirectory named Scripts.  In the Scripts subdirectory is a file that can be executed from a command line interface (CLI) named activate.bat.  For example, for the user Ken this will activate the virtual environment (path will vary on your computer):

  • C:\Users\Ken\bc-venv\Sripts\activate

It is recomended to copy this command to a text file for easy access at a later date.  With the virtual environment activated, the command prompt will change to look something like:

  • (bc-venv) C:\Users\Ken>

Note: By executing py getbc.py --copy you can see the proper command to use for your computer.

What can you do with an activated environment?

  1. To install bomcheck from the command line do:   py -m pip install bomcheck
  2. To upgrade bomcheck from the command line do:   py -m pip install --upgrade bomcheck
  3. To install bomcheckgui from the command line do:   py -m pip install bomcheckgui
  4. To upgrade bomcheckgui from the command line do:   py -m pip install --upgrade bomcheckgui
  5. run bomcheck from the command line:   bomcheck
  6. run bomcheckgui from the command line:   bomcheckgui
See bomcheck_help for more information about running bomcheck from the command line.
